Лабораторная работа N1

Используя текстовый документ который мы сделали ранее на лабораторной работе номер 0, требуется реализовать в C++ или Java программу в которой внедрены 4 метода поиска в отсортированных и неотсортированных массивах по 
ключевому полю в текстовом файле созданном ранее. Для каждого метода поиска требуется провести анализ среднего теоретического времени 
работы и вывод на экран практического времени поиска. Описать по шагам алгоритмы методов поиска.
